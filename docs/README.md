## 项目概述

出版物智能监管一体化平台是面向新闻出版监管部门的综合性政务监管系统，采用"AI基础设施层 + 业务应用层"双层架构：

- **业务应用层**：智能辅助鉴定平台（6模块）、出版物流转监管平台（5模块）、智能舆情分析平台（7模块）
- **AI基础设施层**：智能辅助提升平台（11个AI能力模块）

## 技术栈

| 层次 | 技术选型 |
|------|---------|
| 前端 | Vue 3 + TypeScript + Vite + Ant Design Vue + Pinia + ECharts |
| 后端 | Spring Boot 3 + Spring Cloud + MyBatis-Plus + Sa-Token |
| 数据库 | MySQL 8.0 + Redis 7 + Milvus + Neo4j + Elasticsearch |
| AI | PaddleOCR + vLLM + FunASR + CosyVoice + SadTalker |
| 中间件 | Kafka + MinIO + Nacos + XXL-Job |
| 部署 | Docker + Kubernetes + Helm + Prometheus + Grafana |

## 编码启动顺序建议

1. **第1-2周**：项目脚手架搭建、数据库初始化、基础框架（用户/权限/菜单）
2. **第3-4周**：智能辅助鉴定平台基础功能（出版物管理、任务管理）
3. **第5-6周**：鉴定流程、OCR集成、样本管理
4. **第7-8周**：出版物流转监管平台（商家、资质、物流）
5. **第9-10周**：预警中心、黑白名单、监管大屏
6. **第11-12周**：智能舆情分析平台（采集、分析、报告）
7. **第13-14周**：AI中台核心能力（知识库、智能体、数字人）
8. **第15-16周**：AI中台扩展能力（审校、涉密、翻译）
9. **第17-18周**：联调测试、性能优化、上线部署
