## 1. 设计规范

### 1.1 设计原则

1. **一致性**：全站统一的视觉风格、交互模式、组件库
2. **简洁性**：信息层级清晰，操作路径最短，避免视觉噪音
3. **响应式**：适配主流分辨率（1920×1080、1440×900、1366×768）
4. **可访问性**：符合WCAG 2.1 AA标准，支持键盘导航
5. **性能优先**：首屏加载≤2s，路由切换≤500ms

```
│          │  首页 / 鉴定任务管理 / 任务列表              [刷新] │ ← 面包屑（48px）

│    物流追踪│                                                  │
│    预警中心│                                                  │
│  ▸ 舆情分析│                                                  │
│    采集任务│                                                  │
│    舆情列表│                                                  │

│    菜单    │                                                  │
└──────────┴──────────────────────────────────────────────────┘
   侧边栏（220px，可折叠至80px）        内容区（自适应）
```

### 2.2 顶栏设计

**顶栏左侧**：
- 折叠按钮（菜单展开/收起）
- 面包屑导航

**顶栏中间**：
- 全局搜索框（支持出版物、任务、商家、舆情等搜索）

**顶栏右侧**：
- 通知图标（带未读数量徽标）
- 帮助图标
- 全屏图标
- 用户头像下拉菜单（个人信息、修改密码、退出登录）

### 2.3 侧边栏设计

- 支持折叠/展开
- 支持多级菜单（最多3级）
- 当前菜单高亮显示
- 父菜单自动展开
- 菜单图标 + 文字
- 支持菜单搜索

### 2.4 内容区设计

- 顶部：页面标题 + 操作按钮区
- 中部：搜索表单（可折叠）
- 主体：数据表格/卡片/图表
- 底部：分页（如适用）

### 2.5 响应式断点

| 断点 | 宽度 | 布局调整 |
|------|------|---------|
| xl | ≥1920px | 侧边栏展开，内容区最大化 |
| lg | ≥1200px | 侧边栏展开 |
| md | ≥996px | 侧边栏默认折叠 |
| sm | ≥768px | 侧边栏抽屉模式 |
| xs | <768px | 移动端布局 |
